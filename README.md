# React Scaffold Kit

A CLI tool for scaffolding actions and reducers for redux.

## Installation

`npm i -g rsk`

## Using it

```
rsk <reducer name> <any number of action names>
```

## Config

React Scaffold Kit can be configured using a `.rsk` file. Just add it to your root folder.

```
{
  "style": "ducks",
  "codePath": "src"
}
```
